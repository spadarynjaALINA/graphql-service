# graphql-service
## Installation

```bash
$ npm install
$ cd graphql-serivce
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## Open \`http://localhost:${your port}/graphql\`
```bash
 http://localhost:3000/graphql
```
